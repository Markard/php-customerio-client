# php-customerio-client

PHP client for [Customer.io REST API](https://customer.io/docs/api/).

## Installation

### With Composer

```
$ composer require markard/php-customerio-client
```

```json
{
    "require": {
        "markard/php-customerio-client": "^0.1"
    }
}
```
